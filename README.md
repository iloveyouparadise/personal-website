# Personal Resume Website

This is a personal portfolio website for Tang Ye, showcasing video editing and new media operations skills.

## Getting Started

To run the development server:

```bash
npm start
```

This will start a local server at `http://localhost:8080` (or similar), serving the static files.

## Project Structure

- `index.html`: Main HTML file
- `styles.css`: Stylesheet
- `script.js`: JavaScript for interactivity
- `fonts/`: Font files
- `img/`: Images

## Technologies Used

- HTML5
- CSS3
- JavaScript
